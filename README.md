## __3. Внутри локального json создать файл “new.json”:__
```
 touch new.json
```  

## __4. Добавить файл под гит:__
 ```
 $ git add new.json
 warning: LF will be replaced by CRLF in new.json.
 The file will have its original line endings in your working directory
```

